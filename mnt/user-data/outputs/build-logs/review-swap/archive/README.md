# Review Swap Archive

Closed monthly threads, kept as a permanent record of every swap completed. This is also the source data behind the [Reviewers Board](../../README.md#-reviewers-board) in the main README.

Each file is a snapshot of one month's thread at close: who posted, who reviewed, and the links to both.

## Format

Each archived month follows this structure:

```markdown
# Review Swap — [Month Year]

Closed: [date]
Participants: [count]
Swaps completed: [count]

| Posted by | PR | Reviewed by | Review |
|---|---|---|---|
| @handle | link | @handle | link to review comment |
```
